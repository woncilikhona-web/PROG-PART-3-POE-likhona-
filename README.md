CyberLikh
Project Overview

CyberLikh is a desktop cybersecurity awareness chatbot designed to educate users about online security through intelligent conversation and interactive learning. Developed using C#, Windows Presentation Foundation (WPF), and the .NET 8 framework, the application combines modern user interface design with cybersecurity education to create an engaging learning experience.

Project Objectives

CyberLikh aims to improve cybersecurity awareness by helping users understand common cyber threats and the measures required to protect themselves online. The application demonstrates the use of object-oriented programming principles while implementing secure user authentication, intelligent chatbot behaviour, and conversation management.

Development Phases
Part 1

The first development phase established the chatbot's basic functionality by introducing users to cybersecurity concepts through predefined messages, greetings, and educational information. This phase focused on creating the foundation of the chatbot experience.

Part 2

The second phase introduced a graphical user interface using Windows Presentation Foundation. User authentication was implemented through password creation and verification, while application navigation and usability were significantly improved.

Part 3

The final phase transformed CyberLikh into an intelligent chatbot capable of recognising cybersecurity topics, maintaining conversational context, generating personalised responses, detecting user sentiment, storing previous conversations, and delivering random educational responses. These improvements enhanced both the functionality and realism of the application.

Key Features

CyberLikh includes secure password management with confirmation and password visibility controls. Returning users are able to authenticate using previously stored credentials before accessing the chatbot.

The chatbot recognises discussions relating to phishing, password security, malware, scams, privacy, ransomware, social engineering, public Wi-Fi security, and general cybersecurity awareness. Responses are selected dynamically to provide users with varied and informative conversations.

The application remembers user preferences and maintains conversational context throughout each interaction. Emotional expressions such as frustration, confusion, curiosity, and concern are recognised to provide more appropriate and supportive responses.

Conversation histories may be saved locally for future reference, while a modern user interface featuring animated components and intuitive navigation enhances the overall user experience.

Technologies Used

CyberLikh was developed using C#, Windows Presentation Foundation (WPF), Microsoft Visual Studio 2022, and the .NET 8 framework.

System Requirements

The application requires Windows 10 or Windows 11, Microsoft Visual Studio 2022, and the .NET 8 SDK to build and execute successfully.

Running the Application

Open the solution in Microsoft Visual Studio 2022, restore all required packages if prompted, build the solution, and execute the application. New users should create a password before accessing the chatbot, while returning users authenticate using their previously stored password.

Educational Purpose

CyberLikh was developed as an academic project demonstrating software engineering principles, object-oriented programming, cybersecurity education, user authentication, intelligent chatbot behaviour, and graphical user interface design within a desktop application.

Version History
